    Forkify

    Forkify is a vanilla JavaScript application that interacts with the food2fork API to fetch and display recipe food data. This app uses modern JavaScript tools, such as Webpack to bundle the modules, and Babel to convert ES6, ES7 and ES8 back to ES5. The user can search for a specific recipe, and add ingredients to a shopping list or save to a favourites list via local storage. I created a simple Express server in order to host the app via Heroku. Please refer to my GitHub repo for more information about my additional and re-factored features.

    My Budget Pal

    I created a full-stack application, which allows a user to create an account and log income/expense receipts. The user can categorise receipts and monitor the current budget available. The Express app includes a MongoDB database and user authentication via PassportJS. Receipts can also be edited and deleted by the user. The mobile responsive app is styled with CSS Flexbox.

    Natours

    Natours is a landing page for a fictional company that offers tours for the outdoors. This app was built using pure CSS, including many modern and up to date features. The layout is fully responsive for multiple screen widths. This is my first project using Sass, which is a great extension of CSS that adds power and elegence to the language. The classes are named using the BEM system. I decided to refactor the original code by adding additional variables, mixins etc. This has made the code more maintainable not only for this project, but also re-usable for future Sass projects.

    YelpCamp

    YelpCamp is a RESTful full-stack application, built with ExpressJS, PassportJS and MongoDB. The app uses EJS templates to interact between the backend and frontend. Users can create an account and post, edit and delete campgrounds. User can also review campgrounds and leave comments. I have incorporated the Google Maps API to display a campground's location. The app is styled with Bootstrap.
